# Get-VSS3BucketList

## SYNOPSIS
Gets the list of S3 bucket names, owners and creation dates.

## SYNTAX

```
Get-VSS3BucketList [[-ProfileName] <String>]
```

## PARAMETERS

### -ProfileName
The name of the configuration profile to deploy the stack with.
Defaults to $env:AWS_PROFILE, if set.
