# Import-Vaporshell

## SYNOPSIS
Allows you to import an existing CloudFormation template as a starting point.

## SYNTAX

```
Import-Vaporshell [-Path] <String>
```

## DESCRIPTION
Allows you to import an existing CloudFormation template as a starting point.

## PARAMETERS

### -Path
