# Add-VSBudgetsBudgetCostTypes

 [[-IncludeTax] <Boolean>] [[-IncludeSubscription] <Boolean>] [[-UseBlended] <Boolean>]
 [[-IncludeUpfront] <Boolean>] [[-IncludeDiscount] <Boolean>] [[-IncludeCredit] <Boolean>]
