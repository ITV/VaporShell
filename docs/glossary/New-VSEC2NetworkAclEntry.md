# New-VSEC2NetworkAclEntry

## SYNOPSIS
Adds an AWS::EC2::NetworkAclEntry resource to the template.
Specifies an entry, known as a rule, in a network ACL with a rule number you specify.
Each network ACL has a set of numbered ingress rules and a separate set of numbered egress rules.

## SYNTAX

```
New-VSEC2NetworkAclEntry [-LogicalId] <String> -CidrBlock <Object> [-Egress <Boolean>] [-Icmp <Object>]
 [-Ipv6CidrBlock <Object>] -NetworkAclId <Object> [-PortRange <Object>] -Protocol <Int32> -RuleAction <Object>
 -RuleNumber <Int32> [-DeletionPolicy <String>] [-DependsOn <String[]>] [-Metadata <Object>]
 [-UpdatePolicy <Object>] [-Condition <Object>] [<CommonParameters>]
```

## DESCRIPTION
Adds an AWS::EC2::NetworkAclEntry resource to the template.
