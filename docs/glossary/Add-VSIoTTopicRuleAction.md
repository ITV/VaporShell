# Add-VSIoTTopicRuleAction

## SYNOPSIS

 [[-DynamoDBv2] <Object>] [[-Elasticsearch] <Object>] [[-Firehose] <Object>] [[-Kinesis] <Object>]
 [[-Lambda] <Object>] [[-Republish] <Object>] [[-S3] <Object>] [[-Sns] <Object>] [[-Sqs] <Object>]
