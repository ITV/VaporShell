# Update-VSResourceFunctions

## SYNOPSIS
Updates the Resource and Property Type functions

## SYNTAX

```
Update-VSResourceFunctions [[-Region] <String>] [<CommonParameters>]
```

## DESCRIPTION
Updates the Resource and Property Type functions

## PARAMETERS
