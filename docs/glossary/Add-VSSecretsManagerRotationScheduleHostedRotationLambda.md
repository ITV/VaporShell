# Add-VSSecretsManagerRotationScheduleHostedRotationLambda

## SYNOPSIS
Adds an AWS::SecretsManager::RotationSchedule.HostedRotationLambda resource property to the template.

## SYNTAX

```
Add-VSSecretsManagerRotationScheduleHostedRotationLambda [-RotationType] <Object>
 [[-RotationLambdaName] <Object>] [[-KmsKeyArn] <Object>] [[-MasterSecretArn] <Object>]
 [[-VpcSecurityGroupIds] <Object>] [[-MasterSecretKmsKeyArn] <Object>] [[-VpcSubnetIds] <Object>]
 [<CommonParameters>]
```

## DESCRIPTION
Adds an AWS::SecretsManager::RotationSchedule.HostedRotationLambda resource property to the template.
