# Add-VSMediaPackageOriginEndpointHlsPackage

 [[-PlaylistWindowSeconds] <Object>] [[-PlaylistType] <Object>] [[-AdMarkers] <Object>]
 [[-AdTriggers] <Object>] [[-AdsOnDeliveryRestrictions] <Object>] [[-ProgramDateTimeIntervalSeconds] <Object>]
