# Add-VSS3BucketRedirectRule

## SYNOPSIS

## SYNTAX

```

 [[-ReplaceKeyPrefixWith] <Object>] [[-ReplaceKeyWith] <Object>] [<CommonParameters>]
```

## DESCRIPTION
