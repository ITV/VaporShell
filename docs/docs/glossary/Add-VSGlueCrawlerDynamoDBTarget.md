# Add-VSGlueCrawlerDynamoDBTarget

## SYNOPSIS
Adds an AWS::Glue::Crawler.DynamoDBTarget resource property to the template.
Specifies an Amazon DynamoDB table to crawl.


## DESCRIPTION
